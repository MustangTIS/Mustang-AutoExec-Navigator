==========================================================
 Mustang AutoExec Navigator v1.5
==========================================================

【概要】
Windowsの「自動起動項目」を横断的にスキャンし、一括管理・調査するための
管理者向けナビゲーションツールです。

サービス、タスクスケジューラ、レジストリ、スタートアップフォルダといった
バラバラな場所に存在する起動項目を1つの画面で俯瞰できます。

【主な機能】
1. ノイズ除去フィルタ（起動時に選択可能）
   Microsoft標準の項目を除外し、ユーザーが追加した項目のみを抽出できます。
2. レジストリ一撃ジャンプ
   項目を選択して[OK]を押すと、レジストリエディタが該当キーを直接開きます。
3. 自動Google検索（サービス・タスク）
   正体不明の項目を選択すると、ブラウザでその名前を自動検索します。
4. 連続操作対応（ループ構造）
   調査が終わるたびに一覧に戻るため、複数の項目を連続してチェック可能です。

【ファイル構成】
・AutoRun.bat            : 起動用バッチファイル（右クリックから管理者として実行推奨）
・StartAppNavigator.ps1  : ツール本体（PowerShellスクリプト）

【使い方】
1. 「AutoRun.bat」を実行します。
2. 黒い画面（コンソール）でモードを選択します。
   [1] User Items Only : 自分で入れたソフトだけを見たいとき（おすすめ）
   [2] Show Everything : システム項目を含め、全てを確認したいとき
3. 表示された表から、調査したい項目を選んで [OK] を押します。
   - レジストリ/フォルダなら、その「場所」が開きます。
   - サービス/タスクなら、「Google検索」が走り、管理画面が開きます。
4. 終了する場合は、表をそのまま閉じるか [Cancel] を押してください。

【動作環境】
・Windows 10 / 11 / Windows Server 2011～2025
・PowerShell 5.1 以上
