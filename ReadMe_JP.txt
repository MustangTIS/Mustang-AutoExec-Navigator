==========================================================
 Mustang AutoExec Navigator v1.6.2
==========================================================

【概要】
Windowsの「自動起動項目」を横断的にスキャンし、一括管理・調査するための
管理者向けナビゲーションツールです。

サービス、タスクスケジューラ、レジストリ、スタートアップフォルダといった
バラバラな場所に存在する起動項目を、1つの画面で俯瞰できます [cite: 16]。

【主な新機能 (v1.6.2)】
1. デジタル署名（発行元）の検証
   実行ファイルから「Microsoft」や「Adobe」などの発行元情報を自動取得します。
   署名がない項目（Unsigned）を特定し、不審なプログラムの発見を助けます。
2. 実行権限（RunAs）の可視化
   その項目が「SYSTEM」権限で動いているのか、ユーザー権限なのかを表示します。
   停止させた際の影響範囲を直感的に判断可能です。
3. 日本語インターフェースの完全最適化
   列名を「システム項目」「状態」「実行権限」など、直感的な日本語に統一しました。

【基本機能】
・ノイズ除去フィルタ：Microsoft標準項目を除外し、ユーザー追加分のみを抽出可能 [cite: 16]。
・レジストリ一撃ジャンプ：選択した項目のレジストリキーを直接開きます [cite: 16]。
・自動Google検索：発行元情報を含めた精密なキーワードで正体不明の項目を調査します [cite: 16]。
・連続操作対応：調査が終わるたびに一覧に戻るため、連続チェックが可能です [cite: 16]。

【ファイル構成】
・AutoRun.bat               : 起動用バッチ（必ず「管理者として実行」してください）
・StartAppNavigator_JP.ps1  : 日本語版スクリプト
・StartAppNavigator_EN.ps1  : 英語版スクリプト

【使い方】
1. 「AutoRun.bat」を右クリックし、「管理者として実行」します。
2. 言語（1.日本語 / 2.英語）を選択します。
3. モードを選択します（[1] ユーザー項目のみ推奨）。
4. 表示された表から項目を選んで [OK] を押すと、調査・移動が始まります。

【動作環境】
・Windows 10 / 11 / Windows Server 2011～2025 [cite: 16]
・PowerShell 5.1 以上 [cite: 16]
